# Hybrid调试指南
>
开发接口过程中如何自测接口?

## Dependency
>
- Chrome
- Android SDK Level >=19

## Usage

>
- Android设备连接电脑，并打开USB调试
- 通过Hybrid打开任意页面，测试页面-Slideplus素材中心页
- 使用Chrome浏览器打开，[chrome://inspect](chrome://inspect)
